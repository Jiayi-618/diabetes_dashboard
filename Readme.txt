# Diabetes Health Indicators Dashboard

This project explores how different health indicators, such as BMI, physical activity, smoking status, and age, are associated with diabetes prevalence among U.S. adults using 2015 BRFSS survey data.

## Dataset
- **Source:** [Kaggle - Diabetes Health Indicators Dataset](https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset)
- **Sample Size:** 253,680 individuals
- **Collection Method:** Survey (BRFSS 2015)

## Project Importance
This dashboard highlights key risk factors for diabetes and provides interactive insights for public health interventions.
